# 字体文件说明

请将中文字体文件放置于此目录。

## 推荐字体

| 文件名 | 来源 | 说明 |
|--------|------|------|
| `SimHei.ttf` | Windows 系统字体 | 黑体，图表中文显示 |
| `SimSun.ttf` | Windows 系统字体 | 宋体（备选） |

## 在 Windows 获取字体

字体文件通常位于 `C:\Windows\Fonts\` 目录下。

## 在 macOS 获取

系统内置 `PingFang SC`，代码中已自动检测。

## 注意事项

- 字体文件因版权原因不包含在仓库中
- 确保 `fonts/SimHei.ttf` 存在，否则中文将以英文字体降级显示
